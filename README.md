# Wise World Transfer Bank

Global money transfer platform — send, receive, and exchange money worldwide.

## Tech Stack
- Frontend: Pure HTML/CSS/JS (single file)
- Database: Supabase (PostgreSQL)
- Auth: Supabase Auth
- Deployment: Vercel

## Features
- 💱 Live exchange rates (150+ currencies)
- ₿ Crypto deposit (USDT BEP-20 & TRC-20)
- 🏦 International transfers (SWIFT, Western Union, Wise, MoneyGram)
- 🔐 Email authentication
- 📊 User dashboard

## Deploy
Push to GitHub → Connect to Vercel → Live in 60 seconds.
